# Implement and Verify

1. Read `AGENTS.md`, `docs/architecture.md`, and the target spec.
2. Implement in small increments.
3. Run relevant tests/checks for the changed area.
4. Self-review for regressions and edge cases.
5. Update `TODO.md`, `specs/`, and `docs/quality.md`.
6. Record follow-ups if scope is intentionally deferred.
